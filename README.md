## 🚀 Featured Project — DeviceSense

> A Windows desktop system-monitoring application built from the ground up — **WPF / .NET 8**

| | |
|---|---|
| 🔧 **Hardware monitoring** | Real-time CPU, GPU, storage stats via LibreHardwareMonitorLib |
| 🤖 **AI diagnostics** | Integrated with Google Gemini API for intelligent insights |
| 💾 **Data layer** | EF Core + SQLite, live-updating charts with LiveChartsCore |
| 🌐 **LAN Admin module** | Custom `HttpListener` service for remote device monitoring across a network |
| 👥 **Role** | Backend Lead & System Architect — leading a 4-person team |
